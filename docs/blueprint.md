# **App Name**: UMKM Track

## Core Features:

- User Authentication: Secure user login and registration.
- UMKM Profile: Manage and display UMKM profile information.
- Income Recording: Record income transactions with details.
- Expense Recording: Record expense transactions with details.
- Payment QR Generation: Generate a static QR code for payment.
- Payment Proof Upload: Enable users to upload payment proofs.
- Transaction History: Display a detailed transaction history.
- Financial Reports: Generate simple financial reports (daily & monthly).
- Dashboard: Display a concise dashboard overview.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke trust and security.
- Background color: Light blue (#E0F2F7) to keep the app calm.
- Accent color: Soft blue (#77B5FE) for highlighting important UI elements.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for headlines and body text.
- Simple and professional icons related to finance and business.
- Clean and intuitive layout for easy navigation.
- Subtle animations to provide feedback on user actions.
